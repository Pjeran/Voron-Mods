# UnKlickyBolt
 
UnKlickyBolt is an attempt to make the SexBolt mod for z-Offset a bit eaiser.

Needs:
(3) 3x6 magnets
(1) M3x12 button head screw

Ideally an M3 tap is helpful

Based on the work by Hartk and those before him
